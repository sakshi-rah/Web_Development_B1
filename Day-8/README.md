CSS syntax defines how styles are written.

It tells the browser:
- Which element to style
- What style to apply

CSS works using rules.

Basic CSS Rule Structure:
selector {
property: value; => color: blue;
}

Selector decides which HTML element is targeted.
Property decides what to change.
Value decides how to change it.

Selectors are used to target HTML elements.

Different selectors are used for:
- Single element
- Multiple elements
- Specific elements

Selectors Covered Today:
- Element selector: All same HTML tags
- Class selector: Multiple elements
- ID selector: One unique element
- Group selector: Multiple selectors together

## Priority: Inline>Class>Id>Element
