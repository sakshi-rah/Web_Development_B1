**A form is used to collect data from users.**

Forms are used everywhere:
- Login page
- Registration page
- Contact us page

Form data is usually sent to a server (backend).

Without forms, users cannot interact with websites.

Why Forms are Important?
- Allow user input
- Connect frontend with backend
- Used in almost every real website
- Very important for interviews

Form Related Tags and Attributes Covered:
- form: Form container
- input: User input field
- label: Describes input field
- name: Key sent to backend
- required: Mandatory field
- type: Input type (text, email, etc.)
- placeholder: Hint text inside input
