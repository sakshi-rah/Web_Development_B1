**Lists are used to show items in an ordered or unordered way.**

They help in:
- Showing menus
- Displaying steps
- Grouping related items

Types of Lists:
- Unordered List (ul): Items without order (bullets)
- Ordered List (ol): Items with order (numbers)
- List Item (li): Each item in the list

Tables are used to display structured data in rows and columns.

Tables are commonly used for:
- Marksheets
- Reports
- Pricing data

Table Tags Overview:
- table: Table container
- caption: Table title
- thead: Table header section
- tbody: Table body section
- tfoot: Table footer section
- tr: Table row
- th: Header cell
- td: Data cell
