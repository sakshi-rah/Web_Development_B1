**Links allow users to move from one page to another.**

Links can open:

- Another website
- Another page of the same website
- Email or phone apps

Without links, the web would be just static pages.

Images make websites:

- More attractive
- Easy to understand
- User-friendly

Images are loaded using a path (location).

Tags and Attributes Covered Today:

- a: Anchor tag (link)
- href: Destination of the link
- target: Where to open the link
- img: Display image
- src: Image path
- alt: Alternate text
- link rel="icon": Favicon eg: (link rel="icon" href="./favicon.ico" type="image/x-icon")
