Color is used to:
- Make content readable
- Create visual hierarchy
- Improve user experience

CSS provides multiple ways to define colors.

Color Formats in CSS:
- Color name: red, blue (Simple and basic)
- HEX: #ff0000 (Most commonly used)
- RGB: rgb(0,0,0) (Dynamic colors)
- RGBA: rgba(0, 0, 0, 0.22) (Transparency)
- RGB: rgb(255, 255, 255)
- HSL: hsl(0,100%,50%) (Design systems)

Units define size and spacing.

Used for:
- Font size
- Width and height
- Margin and padding

Common CSS Units:
- px: Fixed pixel
- %: Relative to parent
- em: Relative to parent font size
- rem: Relative to root (html) font size
- vh / vw: Viewport height and width

Includes:
- Font size
- Font weight
- Line height
- Text alignment - center, left, right, end, start
