Flexbox is a CSS layout model.

It helps align and distribute items in one direction (row or column).

Used for:
- Navigation bars
- Cards layout
- Buttons alignment
- Centering elements (very common)

Flexbox solves most alignment problems easily.

Important Flexbox Concepts:
- Flex Container: Parent element with display: flex
- Flex Items: Direct children of flex container

Flexbox Properties Covered:
- display: flex (activate flexbox)
- flex-direction (row or column)
- justify-content (horizontal alignment): start, center, end, space-around, space-batween, space-evenly
- align-items (vertical alignment): Flex-start, Flex-end, Center
- gap (space between items)
