## CSS stands for Cascading Style Sheets.

CSS is used to design and style web pages.

CSS controls:
- Colors
- Fonts
- Spacing
- Layout

HTML creates structure.
CSS makes it beautiful.

Role of CSS in Web:
- Separates content from design
- Makes websites attractive
- Makes websites consistent
- Makes websites responsive
- Allows one design to be applied to many pages

Without CSS:
- Websites look plain
- Hard to maintain design

HTML vs CSS Responsibility:
HTML: Structure
CSS: Styling
HTML: Content
CSS: Design
HTML: Text, forms
CSS: Colors, layout
HTML: Buttons exist
CSS: Buttons look good

Industry Rule:
HTML = WHAT
CSS = HOW IT LOOKS

Types of CSS:
- Inline CSS: Inside HTML tag
- Internal CSS: Inside (style) tag
- External CSS: Separate .css file
